# サポート / Support

## よくある質問 / FAQ

### Q1. スケジュールが保存されません / Schedules are not being saved

**日本語**:
- アプリを再起動してみてください
- デバイスのストレージ容量を確認してください
- それでも解決しない場合は、お問い合わせください

**English**:
- Try restarting the app
- Check your device's storage capacity
- If the issue persists, please contact us

### Q2. カレンダーと同期できません / Cannot sync with calendar

**日本語**:
- デバイスの「設定」→「ScheduleApp」→「カレンダー」のアクセス許可を確認してください
- カレンダーアプリが最新版に更新されているか確認してください
- アプリを再起動してみてください

**English**:
- Check calendar access permission in Settings → ScheduleApp → Calendar
- Confirm that your Calendar app is updated to the latest version
- Try restarting the app

### Q3. 通知が届きません / Notifications are not working

**日本語**:
- デバイスの「設定」→「ScheduleApp」→「通知」が有効になっているか確認してください
- 通知時刻が正しく設定されているか確認してください
- おやすみモードが有効になっていないか確認してください

**English**:
- Check that notifications are enabled in Settings → ScheduleApp → Notifications
- Verify that notification times are set correctly
- Ensure Do Not Disturb mode is not active

### Q4. 写真が添付できません / Cannot attach photos

**日本語**:
- デバイスの「設定」→「ScheduleApp」→「写真」のアクセス許可を確認してください
- 写真アプリに画像があるか確認してください
- デバイスのストレージ容量を確認してください

**English**:
- Check photo access permission in Settings → ScheduleApp → Photos
- Confirm that photos exist in your Photos app
- Check your device's storage capacity

### Q5. スタンプが表示されません / Stamps are not displayed

**日本語**:
- アプリを最新版に更新してください
- アプリを再起動してみてください
- それでも解決しない場合は、お問い合わせください

**English**:
- Update the app to the latest version
- Try restarting the app
- If the issue persists, please contact us

### Q6. データをバックアップしたい / I want to backup my data

**日本語**:
- 現在、データは端末内にのみ保存されています
- iCloudバックアップを有効にすることで、アプリデータもバックアップされます
- デバイスの「設定」→「Apple ID」→「iCloud」→「iCloudバックアップ」を確認してください

**English**:
- Currently, data is stored only on your device
- Enabling iCloud Backup will also backup app data
- Check Settings → Apple ID → iCloud → iCloud Backup on your device

### Q7. データを別の端末に移行したい / I want to transfer data to another device

**日本語**:
- iCloudバックアップを使用して、新しい端末に復元できます
- 古い端末でiCloudバックアップを有効にしてバックアップを実行してください
- 新しい端末で同じApple IDでサインインし、バックアップから復元してください

**English**:
- You can restore data to a new device using iCloud Backup
- Enable iCloud Backup on your old device and perform a backup
- Sign in with the same Apple ID on your new device and restore from backup

### Q8. カテゴリーを削除したい / I want to delete a category

**日本語**:
- 「設定」タブ→「カテゴリー管理」を開いてください
- 削除したいカテゴリーを左にスワイプして削除できます
- カテゴリーを削除しても、そのカテゴリーに紐付いたスケジュールは削除されません

**English**:
- Open Settings tab → Category Management
- Swipe left on the category you want to delete
- Deleting a category will not delete schedules associated with it

### Q9. 広告を非表示にしたい / I want to hide ads

**日本語**:
- Pro版を購入することで広告を非表示にできます
- アプリ内の「Pro版にアップグレード」ボタンから購入できます
- 一度購入すれば、同じApple IDで使用する全ての端末で広告が非表示になります

**English**:
- You can hide ads by purchasing the Pro version
- Purchase through the "Upgrade to Pro" button in the app
- Once purchased, ads will be hidden on all devices using the same Apple ID

### Q10. 繰り返しイベントの設定方法 / How to set recurring events

**日本語**:
- イベント編集画面で「繰り返し」セクションを開いてください
- 「なし」「毎日」「毎週」「毎月」「毎年」から選択できます
- 繰り返し終了日も設定できます

**English**:
- Open the "Recurrence" section in the event edit screen
- Choose from "None", "Daily", "Weekly", "Monthly", or "Yearly"
- You can also set a recurrence end date
