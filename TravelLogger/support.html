アプリについて

TravelLoggerは、GPS位置情報を使用して移動ルートを自動記録する旅行ログアプリです。

主な機能

  • Automatic recording with GPS location
  • Real-time map display
  • Calculate distance, time, and speed
  • Record altitude data
  • Record calendar: Visual monthly record overview
  • GPX/CSV/PDF export (Pro)
  • GPX import (Pro)

FAQ

Q: Recording doesn't start
Please set location permission to "Always Allow". Settings → TravelLogger → Location → "Always Allow"
Q: Battery consumption concerns
Please stop recording when not in use. Background location tracking consumes battery power.
Q: Is there a recording time limit?
The free version has a 6-hour limit. You can extend it by 3 hours by watching rewarded ads. The Pro version has no limit.
Q: Where is data stored?
All data is stored on your device and is never transmitted externally.
Q: Cannot export GPX files
This is a Pro version feature. Please purchase the Pro version.
