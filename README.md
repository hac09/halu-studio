# privacy-policy
プライバシーポリシー
本アプリ（TravelLogger）は、以下のとおり利用者の情報を取り扱います。

取得する情報
・位置情報（バックグラウンドを含む）
・広告配信に必要な情報
・アプリの利用状況

利用目的
・移動ルートの記録およびGPSログの保存
・広告表示および機能提供
・サービス改善のため

第三者提供
本アプリは、広告配信のために第三者サービス（例：Google AdMob）を利用する場合があります。

データの管理
取得した情報は端末内または適切な方法で管理されます。位置情報およびログデータは端末内にのみ保存され、外部サーバーへの送信は一切行いません。
